## Design Context

### Users

Corporate teams using a shared project and task management app in day-to-day planning and execution workflows.

### Brand Personality

Clean, professional, and trustworthy. The interface should communicate clarity, competence, and calm confidence.

### Aesthetic Direction

Light-mode interface with a refined, modern SaaS feel. Balanced visual hierarchy, clear forms, subtle depth, and purposeful illustration usage on auth screens.

### Design Principles

- Prioritize clarity over decoration in core task flows.
- Keep visual hierarchy obvious with strong typography and spacing rhythm.
- Use supportive illustrations to reduce perceived friction on auth screens.
- Maintain fast, responsive interactions with accessible focus and error states.
- Preserve consistency between sign-in and sign-up while giving each page a distinct narrative.
